### Requirements:
 * Python
 * numpy
 * scipy
 * matplotlib
 * Git
 * setuptools (optional)
 * pip (optional)


### Simple Install
1. git clone https://github.com/suavecode/SUAVE.git
2. cd SUAVE/trunk
3. python setup.py install

### Developer Install
1. git clone https://github.com/suavecode/SUAVE.git
2. cd SUAVE/trunk
3. python setup.py develop
