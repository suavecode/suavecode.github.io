SUAVE 2.0

The Latest Evolution in Aerospace Design

SUAVE 2.0 incorporates many new changes to meet the needs of industry, academia, and research. SUAVE 2.0 has been rewritten in Python 3 to maintain the cutting edge of capabilities. In this process SUAVE no longer supports any versions of Python 2. We've listed a subset of the many changes in SUAVE since the last release

  • Python 3
  • Better integration to OpenVSP
  • More Flexible AVL integration
  • Under the hood fixes for speed
  • Ramjets, Scramjets, and Rockets/li>
  • Weight Methodologies for eVTOL
  • Fuel Tank CGs
  • Supersonic Fixes
  • Better Fuselage Segmentation
  • PyOptSparse interface
  • Technological factors
  • More regressions testing to support our Verification and Validation efforts
  • Many little fixes

Are you ready to design the future of aerospace?
